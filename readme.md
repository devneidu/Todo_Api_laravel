TODO APP
========

This is a todo app built with laravel api. The idea is to perform CRUD actions on it.

JWT is also used to handle authentication.

**ENDPOINTS**
-------------

* * *

*   ### GET ALL TODOS
    
    Method: GET REQUEST
    
    Endpoint: 'api/todos'
    
*   ### GET ONE TODO
    
    Method: GET REQUEST
    
    Endpoint: 'api/todos/1'
    
*   ### CREATE TODOS
    
    Method: POST REQUEST
    
    Endpoint: 'api/todos'
    
*   ### UPDTAE TODOS
    
    Method: PUT REQUEST
    
    Endpoint: 'api/todos/1'
    
*   ### DELETE TODOS
    
    Method: DELETE REQUEST
    
    Endpoint: 'api/todos/1'
